AWS Personalization Engine

Wipro Technologies BFSI AWS, Python, Machine Learning, Recommendation Systems, Data Analysis, A/B Testing

Built and deployed a behavioural data-driven personalisation engine on AWS, enabling the client to deliver highly relevant, individualised experiences to users at scale — driving measurable improvements in engagement, retention and conversion rates.

Leveraged AWS machine learning services to build collaborative filtering and content-based recommendation models, trained on rich behavioural datasets including clickstream data, transaction history and user preference signals.

Designed a real-time data pipeline integrating user interaction events with the recommendation engine, ensuring recommendations remained fresh and contextually relevant as user behaviour evolved.

Implemented A/B testing frameworks to continuously evaluate recommendation model performance, enabling data-driven iteration and measurable uplift in key engagement metrics across different user segments.

Led proof-of-concept initiatives that demonstrated the business case for AI-driven personalisation, directly influencing the client's investment in scaling the platform across additional product lines and geographies.
