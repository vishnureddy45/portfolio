Accenture

Conversational AI Chatbot Platform

Accenture Insurance StateFarm Insurance, US Amazon Connect, Amazon Lex, AWS Lambda, CloudWatch, S3, DynamoDB, Salesforce CRM

Designed and implemented an enterprise-grade Conversational AI chatbot on Amazon Connect, enabling customers to resolve queries through natural language interactions and seamlessly escalate to live agents — significantly improving customer satisfaction and reducing agent workload across the contact center.

Built end-to-end chat functionality integrated directly into client-facing web pages via a chat widget, providing real-time AI-powered assistance across the full customer journey from first contact to resolution.

Configured Amazon Lex with advanced NLP capabilities — intent recognition, slot filling and context management — enabling the virtual assistant to handle complex, multi-turn conversations with high accuracy and minimal fallback rates.

Leveraged AWS Lambda for backend business logic and API integrations, enabling real-time retrieval of customer account data and delivering personalised, context-aware bot responses that felt genuinely helpful rather than scripted.

Implemented intelligent escalation workflows within Amazon Connect that smoothly transition customers from the AI assistant to live agents, preserving full conversation context for a seamless, frustration-free handoff experience.

Set up CloudWatch monitoring and logging to proactively track system performance, detect anomalies and continuously improve bot accuracy — delivering measurable reductions in FAQ-related agent interactions and improved first-contact resolution.
